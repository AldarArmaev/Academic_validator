# Интеллектуальный Нормоконтролёр

Система автоматизированного анализа учебных работ (курсовых и дипломных проектов) на базе RAG-архитектуры и детерминированных правил.

## 📋 О проекте

**Интеллектуальный Нормоконтролёр** — это веб-сервис (SaaS-MVP), предназначенный для сокращения времени первичной проверки оформления и содержания учебных работ.

### Целевая аудитория
- Студенты вузов
- Научные руководители

### Основные возможности
- ✅ Проверка форматирования документа (поля, шрифт, интервалы, отступы)
- ✅ Анализ научного стиля текста
- ✅ Верификация ссылок на источники
- ✅ Логический аудит глав работы
- ✅ Анонимизация персональных данных перед обработкой

## 🏗 Архитектура

Система разделена на два независимых вычислительных контура:

### Контур А — Детерминированный валидатор
Отвечает за проверку физических параметров документа:
- Поля страницы (левое 30мм, правое 10мм, верхнее/нижнее 20мм)
- Шрифт Times New Roman, 14pt
- Межстрочный интервал 1.5
- Отступ первой строки 1.25см
- Наличие обязательных разделов и подписей к таблицам/рисункам

**Технологии:** `python-docx`, XML-анализ WordprocessingML

### Контур Б — Семантический анализатор (RAG)
Отвечает за анализ смысла, стиля и верификацию цитат:
- Поиск местоимений первого лица и разговорных оборотов
- Проверка соответствия содержания глав заголовкам
- Верификация ссылок на источники по PDF-файлам

**Технологии:** `sentence-transformers`, `numpy`, LLM API (GPT-4o/Claude)

## 🛠 Стек технологий

| Компонент | Технология |
|-----------|------------|
| Парсинг DOCX | python-docx |
| Парсинг PDF | pdfplumber |
| NER / Анонимизация | Natasha |
| Эмбеддинги | sentence-transformers (paraphrase-multilingual-MiniLM) |
| Векторный поиск | numpy cosine_similarity (MVP), FAISS/ChromaDB (v2+) |
| LLM API | OpenAI GPT-4o / Anthropic Claude |
| Backend | FastAPI + Pydantic |
| Frontend | React + TypeScript |

## 📡 API Endpoints

### `POST /validate`
Полная проверка работы (Контур А + Контур Б).

**Параметры запроса:**
- `work_file` — DOCX файл работы (до 20 страниц)
- `methodology_file` — PDF методички
- `source_files` — PDF источников (до 5 файлов)

**Ответ:** JSON-отчёт с ошибками форматирования, стиля и цитирования.

### `POST /validate/format`
Проверка только форматирования (Контур А).

### `POST /validate/semantic`
Проверка только семантики (Контур Б).

## 🔒 Безопасность и приватность

- **Анонимизация данных:** Все персональные данные (ФИО, кафедры, контакты) заменяются токенами до передачи в LLM-API
- **Временное хранение:** Все файлы и векторные индексы удаляются после завершения сессии
- **On-premise опция:** Поддержка локальной LLM через Ollama + Mistral для полной приватности
